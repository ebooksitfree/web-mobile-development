Book Description
-----------------


    
    If you’re grounded in the basics of Swift, Xcode, and the Cocoa framework, this book provides a structured explanation of all essential real-world iOS app components. Through deep exploration and copious code examples, you’ll learn how to create views, manipulate view controllers, and add features from iOS frameworks.
Create, arrange, draw, layer, and animate views that respond to touchUse view controllers to manage multiple screens of interfaceMaster interface classes for scroll views, table views, text, popovers, split views, web views, and controlsDive into frameworks for sound, video, maps, and sensorsAccess user libraries: music, photos, contacts, and calendarExplore additional topics, including files, networking, and threadsStay up-to-date on iOS 14 innovations, such as:
Multicolor symbol imagesControl action closures and menusTable view cell configuration objectsCollection view lists and outlinesNew split view controller architecturePointer customization on iPadAnd more!Want to brush up on the basics? Pick up iOS 14 Programming Fundamentals with Swift to learn about Swift, Xcode, and Cocoa. Together with Programming iOS 14, you’ll gain a solid, rigorous, and practical understanding of iOS 14 development.
